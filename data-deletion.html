Data Deletion Instructions

GesTurno stores application data locally on the user's device.

Delete Local Data

Users can delete all application data by:

  • Using the application's backup and reset options, if available.
  • Clearing application storage from Android settings.
  • Uninstalling the application.

Google Calendar Data

If Google Calendar synchronization is enabled, users may stop synchronization at any time by disconnecting their Google account from GesTurno.

Calendar events created by GesTurno remain under the user's Google account and can be modified or deleted directly from Google Calendar.
